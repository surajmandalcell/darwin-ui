## shadcn CLI Installation

Install individual components directly into your project:

```bash
# Add a single component
npx shadcn add https://darwin-ui.mandalsuraj.com/registry/button.json

# Add multiple components
npx shadcn add https://darwin-ui.mandalsuraj.com/registry/card.json
npx shadcn add https://darwin-ui.mandalsuraj.com/registry/table.json
```

## Basic Setup

### 1. Import Styles

Add the Darwin UI styles to your application:

```tsx
// In your main entry file (App.tsx, main.tsx, etc.)
import '@smc/darwin-ui/styles';
```

Or in your CSS file:

```css
@import '@smc/darwin-ui/styles';
```

### 2. Use Components

```tsx
import { Button, Card, CardHeader, CardTitle, CardContent } from '@smc/darwin-ui';

function App() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Hello Darwin UI</CardTitle>
      </CardHeader>
      <CardContent>
        <Button variant="primary">Click me</Button>
      </CardContent>
    </Card>
  );
}
```

### 3. Setup Providers (Optional)

For components that require context (Toast, Alert, Modal), wrap your app with providers:

```tsx
import { ToastProvider, AlertProvider, OverlayProvider } from '@smc/darwin-ui';

function App() {
  return (
    <OverlayProvider>
      <ToastProvider>
        <AlertProvider>
          <YourApp />
        </AlertProvider>
      </ToastProvider>
    </OverlayProvider>
  );
}
```

## Peer Dependencies

Darwin UI requires the following peer dependencies:

```json
{
  "react": "^19.0.0",
  "react-dom": "^19.0.0"
}
```

### Optional: Next.js

Some components (Image, MdEditor, ContextMenu) have Next.js-specific features:

```json
{
  "next": ">=14.0.0"
}
```

## Tailwind CSS Configuration

Darwin UI uses Tailwind CSS v4. Ensure your project has Tailwind configured:

```css
/* globals.css */
@import "tailwindcss";
@import "@smc/darwin-ui/styles";
```

## TypeScript

Darwin UI includes TypeScript declarations. No additional `@types` packages needed.

```tsx
import type { Variant, Size } from '@smc/darwin-ui';

const buttonVariant: Variant = 'primary';
const buttonSize: Size = 'lg';
```
