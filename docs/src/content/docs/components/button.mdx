## Usage

```tsx title="src/components/MyComponent.tsx"
import { Button } from '@smc/darwin-ui';

export function Example() {
  return (
    <div className="flex gap-4">
      <Button variant="primary">Save Changes</Button>
      <Button variant="secondary">Cancel</Button>
    </div>
  );
}
```

## Variants

The button component includes 8 distinct variants designed for clear visual hierarchy.

<div class="overflow-x-auto">
  <table>
    <thead>
      <tr>
        <th>Variant</th>
        <th>Description</th>
        <th>Visual Style</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>`default`</td>
        <td>Standard action, neutral priority.</td>
        <td>Glass background, subtle border.</td>
      </tr>
      <tr>
        <td>`primary`</td>
        <td>Main call-to-action on the page.</td>
        <td>Solid blue with ambient glow.</td>
      </tr>
      <tr>
        <td>`secondary`</td>
        <td>Alternative actions.</td>
        <td>Lighter glass effect.</td>
      </tr>
      <tr>
        <td>`destructive`</td>
        <td>Dangerous or irreversible actions.</td>
        <td>Solid red with warning glow.</td>
      </tr>
      <tr>
        <td>`outline`</td>
        <td>Low emphasis actions.</td>
        <td>Transparent with visible border.</td>
      </tr>
      <tr>
        <td>`ghost`</td>
        <td>Minimal buttons, often used in toolbars.</td>
        <td>Transparent, background on hover.</td>
      </tr>
    </tbody>
  </table>
</div>

## Advanced Examples

### With Icons

Buttons support arbitrary React nodes, making them perfect for icon-labels.

<div class="not-content">
  <ButtonWithIconPreview client:load />
</div>

```tsx
<Button variant="primary">
  <Plus className="w-4 h-4 mr-2" />
  Create New
</Button>
```

### Loading State

Set the `loading` prop to `true` to show a spinner and disable interaction.

```tsx
<Button loading>Saving...</Button>
```
