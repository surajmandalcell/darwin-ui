Skeleton is a placeholder component that simulates the layout of loading content.

</div>

## Installation

<Tabs>
  <TabItem label="CLI">
    ```bash
    npx shadcn add https://darwin-ui.mandalsuraj.com/registry/skeleton.json
    ```
  </TabItem>
  <TabItem label="Manual">
    ```bash
    npm install @smc/darwin-ui
    ```
  </TabItem>
</Tabs>

## Usage

```tsx
import { Skeleton } from '@smc/darwin-ui';

export function Example() {
  return (
    <div className="flex items-center space-x-4">
      <Skeleton className="h-12 w-12 rounded-full" />
      <div className="space-y-2">
        <Skeleton className="h-4 w-[250px]" />
        <Skeleton className="h-4 w-[200px]" />
      </div>
    </div>
  );
}
```
