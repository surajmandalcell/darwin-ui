The Input component is a polished form field that supports various types, states, and icon adornments, all with a consistent glass-morphism aesthetic.

## Installation

<Tabs>
  <TabItem label="CLI">

## Usage

```tsx
import { Input, Label } from '@smc/darwin-ui';

export function LoginInput() {
  return (
    <div className="grid w-full max-w-sm items-center gap-1.5">
      <Label htmlFor="email">Email</Label>
      <Input type="email" id="email" placeholder="Email" />
    </div>
  );
}
```

## Examples

### With Icons

Inputs can be adorned with icons on the left or right to provide context.

```tsx
import { Search } from 'lucide-react';

<div className="relative">
  <Search className="absolute left-2.5 top-2.5 h-4 w-4 text-white/50" />
  <Input className="pl-9" placeholder="Search..." />
</div>
```

### Disabled State

```tsx
<Input disabled type="email" placeholder="Email" />
```

### File Input

```tsx
<Input id="picture" type="file" />
```
