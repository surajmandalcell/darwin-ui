---
title: useMobile
description: Shorthand hook for mobile device detection.
---

import { Tabs, TabItem } from '@astrojs/starlight/components';

A convenience hook wrapping `useMediaQuery` for standard mobile breakpoints.

## Installation

```tsx
import { useMobile } from '@smc/darwin-ui';
```

## Usage

```tsx
export function Nav() {
  const isMobile = useMobile();

  return isMobile ? <MobileNav /> : <DesktopNav />;
}
```
