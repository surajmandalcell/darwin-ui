title: Badge

import { Tabs, TabItem } from '@astrojs/starlight/components';

Badges are used to highlight status, tags, or counts. They are designed to be compact and sit alongside broader content.

## Installation

<Tabs>
  <TabItem label="CLI">
    ```bash
    npx shadcn add https://darwin-ui.mandalsuraj.com/registry/badge.json
    ```
  </TabItem>
  <TabItem label="Manual">
    ```bash
    npm install @smc/darwin-ui
    ```
  </TabItem>
</Tabs>

## Usage

```tsx
import { Badge } from '@smc/darwin-ui';

export function Example() {
  return <Badge>Badge</Badge>;
}
```
