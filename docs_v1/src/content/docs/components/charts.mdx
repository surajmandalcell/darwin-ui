## Area Chart

Used for visualizing trends over time, like revenue or traffic.

```tsx
import { AreaChart, Area, XAxis, YAxis, CartesianGrid, Tooltip, ResponsiveContainer } from 'recharts';

const data = [
  { name: 'Jan', value: 4000 },
  { name: 'Feb', value: 3000 },
  { name: 'Mar', value: 2000 },
  { name: 'Apr', value: 2780 },
  { name: 'May', value: 1890 },
  { name: 'Jun', value: 2390 },
];

export function RevenueChart() {
  return (
    <div className="h-[300px] w-full">
      <ResponsiveContainer width="100%" height="100%">
        <AreaChart data={data}>
          <defs>
            <linearGradient id="colorValue" x1="0" y1="0" x2="0" y2="1">
              <stop offset="5%" stopColor="#3b82f6" stopOpacity={0.3}/>
              <stop offset="95%" stopColor="#3b82f6" stopOpacity={0}/>
            </linearGradient>
          </defs>
          <CartesianGrid strokeDasharray="3 3" stroke="rgba(255,255,255,0.05)" vertical={false} />
          <XAxis 
            dataKey="name" 
            stroke="rgba(255,255,255,0.1)"
            tick={{ fill: "rgba(255,255,255,0.4)", fontSize: 12 }} 
            tickLine={false}
            axisLine={false}
          />
          <Tooltip 
            contentStyle={{ 
              backgroundColor: "rgba(10,10,10,0.9)", 
              border: "1px solid rgba(255,255,255,0.1)",
              borderRadius: "8px"
            }}
            itemStyle={{ color: "#fff" }}
          />
          <Area 
            type="monotone" 
            dataKey="value" 
            stroke="#3b82f6" 
            fillOpacity={1} 
            fill="url(#colorValue)" 
          />
        </AreaChart>
      </ResponsiveContainer>
    </div>
  );
}
```

## Styling Guidelines

To maintain the Darwin UI look:

1.  **Grid Lines**: Use `stroke="rgba(255,255,255,0.05)"` and `vertical={false}`.
2.  **Axes**: Remove axis lines with `axisLine={false}` and `tickLine={false}`. Use muted colors for text.
3.  **Tooltips**: Use a custom `contentStyle` with high opacity backing (e.g., `rgba(10,10,10,0.9)`) and a subtle border.
4.  **Gradients**: Always use `defs` to create fade-out gradients for area fills.
